# Visual Kur

## Instalation 
 ``` $ yarn ```

## Run 
  
 ``` $ yarn dev```
  
## Build Release

```  $ yarn build```

## Run Release
  
```  $ yarn electron .```

## Example File
In order to test the application, an example yaml file is provided on the public folder with the name mnist.yml. This file contains the example provided on Kur Github page.
